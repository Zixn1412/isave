# iSave - ระบบแจ้งซ่อมครุภัณฑ์ (PHP + MySQL)
This is a minimal working example to run on XAMPP (place folder under `htdocs/isave`).

## Setup
1. Import `isave_db.sql` in phpMyAdmin.
2. Place this `isave_project` folder under `htdocs/isave`.
3. Update `config.php` DB credentials if needed.
4. Run `init_admin.php` once to create initial admin (then delete the file).
